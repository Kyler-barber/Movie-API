Movie API Documentation

This API provides information about various movies, including titles, directors, genres, and user management functionality.

Request URL Response
GET /movies

Returns a list of all movies.

URL: /movies
Method Type: GET
Query Parameters: None
Request Body: None
Response Format: 
                    [
                      {
                        "title": "The Shawshank Redemption",
                        "year": 1994
                      },
                      {
                        "title": "The Godfather",
                        "year": 1972
                      }
                    ]
GET /movies/:title

Returns data about a single movie by title.

URL: /movies/:title
Method Type: GET
Query Parameters: Title of the movie in the URL
Request Body: None
Response Format: 
                    {
                      "title": "The Godfather",
                      "description": "The aging patriarch of an organized crime dynasty...",
                      "genre": "Crime",
                      "director": "Francis Ford Coppola",
                      "imageURL": "http://example.com/image.jpg",
                      "featured": true
                    }
GET /genres/:name

Returns a description of the genre by name.

URL: /genres/:name
Method Type: GET
Query Parameters: Genre name in the URL
Request Body: None
Response Format: 
                    {
                      "name": "Action",
                      "description": "A genre with fast-paced scenes and high energy."
                    }
GET /directors/:name

Returns the director's bio, birth year, and death year by name.

URL: /directors/:name
Method Type: GET
Query Parameters: Director name in the URL
Request Body: None
Response Format: 
                    {
                      "name": "Francis Ford Coppola",
                      "bio": "American film director and producer...",
                      "birthYear": 1939,
                      "deathYear": null
                    }
POST /users

Registers a new user.

URL: /users
Method Type: POST
Request Body: JSON format:
                    {
                      "username": "johnDoe",
                      "email": "john@example.com"
                    }
Response Format: 
                    {
                      "message": "New user registered."
                    }
PUT /movies/:title/description

Updates the description of a movie by title.

URL: /movies/:title/description
Method Type: PUT
Request Body: JSON format:
                  {
                    "description": "New movie description here."
                  }
Response Format: 
                  {
                    "message": "Description updated for movie: title."
                  }
PUT /directors/:name/bio

Updates the bio of a director for all movies by the director.

URL: /directors/:name/bio
Method Type: PUT
Request Body: JSON format:
                  {
                    "bio": "New director bio here."
                  }
Response Format: 
                  {
                    "message": "Bio updated for director: name."
                  }
