Movie API Documentation

This API provides information about various movies, including titles, directors, genres, and user management functionality.

Request URL Response
GET /movies

Returns a list of all movies.

URL: /movies
Method Type: GET
Query Parameters: None
Request Body: None
Response Format: 
                    [
                      {
                        "title": "The Shawshank Redemption",
                        "year": 1994
                      },
                      {
                        "title": "The Godfather",
                        "year": 1972
                      }
                    ]
GET /movies/:title

Returns data about a single movie by title.

URL: /movies/:title
Method Type: GET
Query Parameters: Title of the movie in the URL
Request Body: None
Response Format: 
                    {
                      "title": "The Godfather",
                      "description": "The aging patriarch of an organized crime dynasty...",
                      "genre": "Crime",
                      "director": "Francis Ford Coppola",
                      "imageURL": "http://example.com/image.jpg",
                      "featured": true
                    }
GET /genres/:name

Returns a description of the genre by name.

URL: /genres/:name
Method Type: GET
Query Parameters: Genre name in the URL
Request Body: None
Response Format: 
                    {
                      "name": "Action",
                      "description": "A genre with fast-paced scenes and high energy."
                    }
GET /directors/:name

Returns the director's bio, birth year, and death year by name.

URL: /directors/:name
Method Type: GET
Query Parameters: Director name in the URL
Request Body: None
Response Format: 
                    {
                      "name": "Francis Ford Coppola",
                      "bio": "American film director and producer...",
                      "birthYear": 1939,
                      "deathYear": null
                    }
POST /users

Registers a new user.

URL: /users
Method Type: POST
Query Parameters: None
Request Body: JSON format:
                    {
                      "username": "johnDoe",
                      "email": "john@example.com"
                    }
Response Format: 
                    {
                      "message": "New user registered."
                    }
PUT /users/:username

Updates a user's username.

URL: /users/:username
Method Type: PUT
Query Parameters: Current username in the URL
Request Body: JSON format:
                    {
                      "newUsername": "johnNew"
                    }
Response Format: 
                    {
                      "message": "User info updated for username: johnDoe."
                    }
POST /users/:username/favorites

Adds a movie to the user's list of favorites.

URL: /users/:username/favorites
Method Type: POST
Query Parameters: Username in the URL
Request Body: JSON format:
                    {
                      "movieTitle": "Inception"
                    }
Response Format: 
                    {
                      "message": "Movie Inception added to favorites for user: johnDoe."
                    }
DELETE /users/:username/favorites/:title

Removes a movie from the user's list of favorites.

URL: /users/:username/favorites/:title
Method Type: DELETE
Query Parameters: Username and movie title in the URL
Request Body: None
Response Format: 
                    {
                      "message": "Movie Inception removed from favorites for user: johnDoe."
                    }
DELETE /users/:email

Deregisters a user by email.

URL: /users/:email
Method Type: DELETE
Query Parameters: Email in the URL
Request Body: None
Response Format: 
                    {
                      "message": "User with email john@example.com has been removed."
                    }
